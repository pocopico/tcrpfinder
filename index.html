How it works: Phase 1 — fast TCP probes across all 254 host addresses (smart-ordered: likely TCRP ranges first). Phase 2 — confirmed-open ports are queried at /api/discover for TCRP, then the root page is fingerprinted for Synology DSM (Web Assistant). Subnet is auto-detected via WebRTC; select a chip or type one manually if detection misses. Requires tcrp-node ≥ 0.11 with the updated server.js (adds Access-Control-Allow-Private-Network: true for Chrome compatibility). DSM detection works on port 5000 — both pre-install (setup wizard) and post-install (login) states are identified.
Chrome users: if the scan finds open ports but no TCRP, use the Test IP box above to pinpoint the issue, or try Firefox. TCRP repo
